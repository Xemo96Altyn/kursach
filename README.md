# Автоматизация приёма абитуриентов

Курсовой проект — десктопное приложение для приёмной комиссии на C++ с SQLite.

## Стек

- C++, .NET Windows Forms
- SQLite3 (System.Data.SQLite)
- Visual Studio 2022

## Возможности

- Хранение и обработка данных абитуриентов
- Поиск и фильтрация заявок
- Генерация отчётов для приёмной комиссии

## Запуск

Открыть `kursach1.sln` в Visual Studio, собрать и запустить.
